For learning & puzzles — these ciphers are not secure encryption and shouldn't be used to protect sensitive information.

Affine Guide

Each letter x becomes (a·x + b) mod m. Only values of a coprime with m are reversible — and since 29 is prime, every a from 1–28 works for the Scandinavian alphabets. Caesar is the special case a = 1.

Playfair Guide

A keyword grid encrypts letter pairs (I and J share a cell). Doubled letters are split and an odd final letter is padded with X, so non-letters are dropped and decoded text keeps that padding. The Scandinavian variants use a 4×7 grid to fit their extra letters.

Polybius Guide

Each letter becomes its row-column pair on the grid. English uses the classic 5×5 (I and J share a cell); the Scandinavian variants keep all 29 letters on a 6×5 grid. Word breaks are shown as "/".

Bacon's Cipher Guide

Each letter becomes a 5-symbol A/B group — its position in the alphabet in binary. Five bits give 32 patterns, enough for the 26 English or 29 Scandinavian letters. Word breaks are shown as "/".

Beaufort Guide

A Vigenère relative using c = (key − plain) mod m. It's reciprocal — encoding twice with the same keyword returns the original, so Encode and Decode do the same thing.

Autokey Guide

Like Vigenère, but the keystream is the keyword followed by the message itself, so the key never repeats — a stronger, classic improvement over plain Vigenère.

Columnar Guide

The message is written in rows under the keyword, then the columns are read out in the alphabetical order of the keyword's letters. Every character — including spaces and Æ/Ø/Å — is moved as-is, so it reverses exactly.

Scytale Guide

The ancient rod cipher: write the message across this many columns, then read down each column. A keyless transposition, so any character round-trips exactly.

Brute Force Guide

Paste intercepted ciphertext and drag the slider until the output reads as plain language — that shift number is the key. Or enable "Show all shifts" to list every candidate at once.

A1Z26 Guide

Each letter becomes its alphabet position, hyphen-separated per word. Decode with the same alphabet the message was encoded with.

Binary Reverse Guide

Letters map in reverse (Z=1 … A=26) with randomized digit substitution. Fixed width packs 5-bit chunks; variable width separates numbers with spaces. Decode with the same width mode the message was encoded with.

Elder Futhark Guide

Transliterates text into the 24-rune Elder Futhark. C, K and Q share ᚲ; V and W share ᚹ; Y uses ᛁ; X becomes ᚲᛊ. TH and NG get their own single runes ᚦ and ᛜ. Scandinavian letters map to Æ → ᛇ, Ø → ᚯ and Å → ᚬ.

Anagram Helper Guide

Scrambles your input text into the output box while preserving spaces. Click the Shuffle icon in the output header to randomize the letters again.
